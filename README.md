# OSCE Mentor AI

An AI-powered clinical exam simulator for Egyptian medical students preparing for OSCE stations.

## Features
- Simulated Patient Interaction (History taking)
- Step-by-step Physical Examination
- Investigation Ordering and Interpretation
- Automated Examiner Feedback and Scoring
- Multilingual Voice Recognition (Arabic + English)

## Setup
1. `npm install`
2. Configure `GEMINI_API_KEY` in your environment.
3. `npm run dev`

## Deployment
Compatible with Replit and Cloud Run.
- Build: `npm run build`
- Start: `npm start`
